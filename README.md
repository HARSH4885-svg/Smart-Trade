# 🚀 Smart Trade

An advanced AI-powered stock trading simulation platform that enables users to practice trading, analyze markets, and make smarter financial decisions — all in a risk-free environment.

---

## 📌 Overview

**Smart Trade** is a modern fintech web application designed to simulate real-world stock trading. It integrates real-time (or simulated) stock data, interactive charts, portfolio management, and AI-driven insights.

Built as part of a hackathon project, this platform showcases a complete trading ecosystem with advanced features like wallet systems, loans, and social trading simulation.

## ✨ Features

### 📊 Trading & Portfolio

* Buy/Sell stocks with live price updates
* Portfolio tracking with real-time profit/loss
* Transaction history system

### 📈 Charts & Analytics

* Interactive charts (1D, 5D, 1M, All-time)
* Real-time stock data integration
* Smooth UI with responsive design

### 🤖 AI Integration

* AI-based stock insights
* Market trend suggestions
* Extendable for trading bots

### 💰 Wallet & Loan System

* Virtual wallet for trading
* Loan feature when balance is insufficient
* Auto-adjustment based on trades

### 🛒 Marketplace Simulation

* Live activity banner (e.g., “User X bought Y stocks”)
* Social trading simulation environment

### 🔐 Authentication

* Secure login/signup
* Firebase authentication system

### ⚙️ Admin / Control Features

* User tracking (extendable)
* Transaction monitoring
* System scalability

## ⚙️ Setup Instructions

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/HARSH4885-svg/Smart-Trade.git
cd Smart-Trade
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Setup Environment Variables

Create a `.env` file in the root directory and add:

```env
VITE_FIREBASE_API_KEY=your_firebase_api_key
VITE_FINNHUB_API_KEY=your_finnhub_api_key
```

---

### 4️⃣ Run the Project

```bash
npm run dev
```
